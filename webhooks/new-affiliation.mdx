## Atributos

Esta é a descrição dos parâmetros enviados pelo webhook em resposta a determinado evento.

| Nome do parâmetro | Descrição | Formato e tamanho suportado |
| --- | --- | --- |
| event | Tipo de evento. Ver os **valores possíveis** abaixo. | String até 255 caracteres |
| status | Status da afiliação. Ver os **valores possíveis** abaixo. | String até 255 caracteres |
| from_auto_affiliation | Indica se a afiliação foi criada automaticamente. Possíveis valores: 0 (Não foi criada automaticamente). | Número inteiro |
| creator_name | Nome do criador da afiliação | String até 255 caracteres |
| creator_email | Email do criador da afiliação | String no formato de email |
| code | Código da afiliação | String até 255 caracteres |
| commission_percentage | Porcentagem de comissão | Número inteiro entre 0 e 100 |
| product_id | Identificador do produto associado à afiliação | Número inteiro com até 20 dígitos |
| product_name | Nome do produto associado à afiliação | String até 255 caracteres |
| affiliate_id | Identificador do afiliado | Número inteiro com até 20 dígitos |
| name | Nome do afiliado | String até 255 caracteres |
| email | Email do afiliado | String no formato de email |
| phone | Telefone do afiliado | String contendo números e caracteres especiais |
| document | CPF do afiliado | String no formato de CPF |
| affiliation_date | Data da afiliação | String no formato de data |

## Valores possíveis

<AccordionGroup>

<Accordion title="event" description="Tipo de evento.">

| Valor | Descrição |
| --- | --- |
| `AFFILIATION_CREATED` | Afiliação Criada |
| `AFFILIATION_REQUESTED` | Afiliação Solicitada |
| `AFFILIATION_APPROVED` | Afiliação Aprovada. |

</Accordion>

<Accordion title="status" description="Status da afiliação.">

| Valor | Descrição |
| --- | --- |
| `approved` | Afiliação Aprovada |
| `pending` | Afiliação Pendente. |

</Accordion>

</AccordionGroup>
