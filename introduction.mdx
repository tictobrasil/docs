A Ticto API v2 é a interface de integração pública da Ticto que permite criar e gerenciar **produtos** e **ofertas** diretamente via código. Com ela, você conecta sua central, ERP ou sistema interno à Ticto sem precisar acessar o painel manualmente para cada operação.

## O que você pode fazer

A API v2 expõe dois recursos principais:

- **Produtos** — crie cursos, ebooks, assinaturas, softwares e produtos físicos
- **Ofertas** — defina preços, formas de cobrança (única ou recorrente) e condições de pagamento para cada produto

Com poucas linhas de código, você envia apenas os campos essenciais. O restante é preenchido automaticamente pelos padrões da Ticto com base no tipo de produto.

## Como funciona

<Steps>
  <Step title="Gere um token de API">
    Acesse o painel da Ticto, navegue até **Integrações → API v2** e crie um token com os escopos necessários.
  </Step>
  <Step title="Autentique suas requisições">
    Passe o token no header `Authorization: Bearer` em cada chamada à API.
  </Step>
  <Step title="Crie produtos e ofertas">
    Use os endpoints `/api/v2/products` e `/api/v2/products/{hash}/offers` para criar e listar seus recursos.
  </Step>
  <Step title="Ative na dashboard">
    Produtos criados via API ficam como **rascunho** por padrão. Ative-os no painel quando estiver pronto para vender.
  </Step>
</Steps>
