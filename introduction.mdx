A Ticto API v2 é a interface de integração pública da Ticto que permite criar e gerenciar **produtos** e **ofertas** diretamente via código. Com ela, você conecta sua central, ERP ou sistema interno à Ticto sem precisar acessar o painel manualmente para cada operação.

## O que você pode fazer

**Escrita**, para criar recursos por código:

- **Produtos**: cursos, ebooks, mentorias, consultorias e outros produtos digitais
- **Ofertas**: preços, forma de cobrança (única ou recorrente) e métodos de pagamento de cada produto

**Leitura**, para consultar seus dados (paridade com a API v1, para quem está migrando):

- **Pedidos**, **Assinaturas** e **Saques**: histórico, resumo e detalhe por recurso

Na escrita, você envia apenas os campos essenciais. Os ~80 campos internos de uma oferta são preenchidos automaticamente pelos padrões da Ticto, a partir de um preset seguro por tipo de produto.

## Como funciona

<Steps>
  <Step title="Gere um token de API">
    Acesse o painel da Ticto, navegue até **Integrações → API v2** e crie um token com os escopos necessários.
  </Step>
  <Step title="Autentique suas requisições">
    Passe o token no header `Authorization: Bearer` em cada chamada à API.
  </Step>
  <Step title="Crie produtos e ofertas">
    Use os endpoints `/api/v2/products` e `/api/v2/products/{hash}/offers` para criar e listar seus recursos.
  </Step>
  <Step title="Comece a vender">
    Produtos criados via API nascem **pré-aprovados** e ativos, prontos para vender. Envie `is_active: false` se quiser prepará-los antes de divulgar e ativá-los depois no painel.
  </Step>
</Steps>
