Liste todas as ofertas associadas a um produto da sua conta. A resposta é paginada e inclui detalhes completos de cada oferta, como preço em centavos, tipo de cobrança, configurações de recorrência e o link direto de checkout.

## Endpoint

```
GET /api/v2/products/{product_hash}/offers
```

## Parâmetros de path

<ParamField path="product_hash" type="string" required>
  Hash único do produto cadastrado na Ticto. Visível na URL do produto no painel.
</ParamField>

## Parâmetros de query

<ParamField query="page" type="integer">
  Número da página a ser retornada. Padrão: `1`.
</ParamField>

<ParamField query="per_page" type="integer">
  Quantidade de registros por página. Padrão: `20`. Máximo: `100`.
</ParamField>

## Requisição

```bash cURL
curl --request GET \
  --url "https://api.ticto.com.br/api/v2/products/PROD-ABC123/offers?page=1&per_page=20" \
  --header "Authorization: Bearer {seu_token}"
```

## Resposta 200

Retorna um array paginado de objetos de oferta no campo `data`, acompanhado de metadados de paginação em `meta`.

```json Resposta 200
{
  "data": [
    {
      "id": "OFRT-8H2K",
      "reference_id": 55190,
      "name": "Plano Mensal",
      "price": 9700,
      "charge_type": "recurring",
      "recurrence": {
        "interval": "monthly",
        "trial_days": 0,
        "cycles": null
      },
      "status": "active",
      "checkout_url": "https://checkout.ticto.com.br/OFRT-8H2K",
      "created_at": "2024-03-15T14:35:00Z"
    }
  ],
  "meta": {
    "current_page": 1,
    "per_page": 20,
    "total": 3
  }
}
```

<Note>
  Somente ofertas de produtos da sua conta são retornadas. Tentativas de listar ofertas de outro produtor retornam `404`.
</Note>

## Erros possíveis

| Status HTTP | Slug do erro         | Descrição                                                              |
|-------------|----------------------|------------------------------------------------------------------------|
| `401`       | `unauthenticated`    | Token ausente, inválido ou expirado.                                   |
| `403`       | `forbidden`          | O token não possui o escopo `offers:read` necessário para esta rota.   |
| `404`       | `product_not_found`  | Nenhum produto com o `product_hash` informado foi encontrado na conta. |
